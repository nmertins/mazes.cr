# mazes.cr

Coding along with [Mazes for Programmers](https://g.co/kgs/GMz8Nr) by Jamis Buck in [Crystal](https://crystal-lang.org/).

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     mazes.cr:
       github: nmertins/mazes.cr
   ```

2. Run `shards install`

## Usage

```crystal
require "mazes.cr"
```
